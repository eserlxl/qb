# Step 4 Implementation Handoff Prompt Template

This reference is not an auto-executed QB planning step.

The Step 4 implementation handoff is single-sourced in
`references/handoffs/run-step4.md` (which carries `contract_version` frontmatter): the
audit-status print gates, the copy block, the per-slice implementation contract, operator
notes, stop gates, and resume/recovery all live there.

Read `references/handoffs/run-step4.md` and print its Copy Block for the user to paste into
a new Antigravity task or conversation — only after Step 3 writes `.qb/sub-planning-audit.md`
and the audit permits Step 4, and subject to the print gates in that file.
